# Immersive EGO: CIVITAS

Immersive EGO: CIVITAS est un projet d'integration GPL-3.0-only pour Minecraft
1.21.1 et NeoForge. Le projet relie MineColonies, MCA Reborn, Modern
Companions et Immersive EGO dans une seule simulation civique.

## Regles de base

- Les dependances restent des JARs externes separes.
- Aucun asset, logo, modele, son, skin, texture, structure ou element graphique
  upstream n'est copie par defaut.
- Tout code GPL copie ou adapte doit etre lie a un commit source immutable et a
  une ligne du manifeste de provenance.
- MCA reste l'autorite de la famille et des relations.
- MineColonies reste l'autorite de la colonie.
- Immersive EGO reste l'autorite des besoins physiques et psychologiques.

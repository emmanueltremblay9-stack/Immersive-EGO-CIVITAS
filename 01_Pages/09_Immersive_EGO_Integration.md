# Immersive EGO integration

Immersive EGO remains a separate MIT project and authority for body and mind.

## Required neutral API

CIVITAS needs an external-subject API capable of:

- resolving/registering a supported NPC subject;
- reading a bounded vital/readiness snapshot;
- querying work, fight, travel and recovery capability;
- submitting civic stimuli;
- supporting unloaded time advancement where available.

## Legal boundary

No GPL-derived CIVITAS class or implementation enters the Immersive EGO repository. The shared API uses neutral interfaces, identifiers and data records.

## Civic uses

- Readiness affects work and military availability.
- Hydration, fatigue and sleep affect schedules.
- Family safety and shelter affect psychological recovery.
- Injury and trauma affect guard duty.
- Origin may influence cultural affinity, but never automatically determines faction.

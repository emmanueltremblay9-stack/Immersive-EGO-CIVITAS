# GPL source provenance policy

1. No copied file enters the repository without a manifest entry.
2. Every derived source file includes an adaptation header.
3. Original path and immutable commit are preserved.
4. Modified versions are clearly marked.
5. Corresponding source is published for every distributed binary.
6. Runtime dependencies are not shaded into the CIVITAS JAR.
7. Asset copying is forbidden unless separately verified and explicitly approved.
8. Logos and trademarks are not treated as ordinary GPL source.
9. Modern Companions files require third-party lineage verification.
10. CI blocks files missing provenance metadata.
